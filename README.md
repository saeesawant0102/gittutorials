# GitTutorials

A simple repository to learn and practice Git basics.

##  About

This repository is created as a hands-on guide to understand how Git works. It includes basic commands, workflows, and examples to help beginners get comfortable with version control.

##  Features

* Basic Git commands
* Branching and merging
* Commit history examples
* Practice files for learning

##  Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/gittutorials.git
   ```

2. Navigate to the project folder:

   ```bash
   cd gittutorials
   ```

3. Start exploring and practicing Git commands.

##  Topics Covered

* `git init`
* `git add`
* `git commit`
* `git branch`
* `git merge`
* `git push` & `git pull`
